# VoiceAgent 🎧🤖

A lightweight Python voice assistant that listens, understands, and speaks — all in real-time and offline.

## ✨ What it does
- 🎙️ Listens to your voice
- 🧠 Uses AI to generate smart replies
- 🔊 Speaks responses back using offline TTS
- 📈 Logs interactions to `call_logs.xlsx`

## 🚀 Run It
```bash
pip install -r requirements.txt
python main.py

📂 Project Structure
main.py – Entry point

agent_pipeline.py – Handles audio → text → response → voice

utils/ – Includes STT, LLM, TTS modules


💡 Uses
Perfect for AI demos, smart assistants, and offline voice agent
